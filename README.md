## 💡 Reflexión sobre el desarrollo y el uso de IA

Para la creación de esta página utilicé Inteligencia Artificial como herramienta de asistencia generativa para estructurar el código base.
El proceso me dejó un aprendizaje muy importante: si bien hoy en día las IA te permiten levantar una interfaz o generar líneas de código de forma directa y rapidísima, **es totalmente fundamental entender qué hace cada etiqueta, función y estilo que se está ejecutando**.

Las herramientas automáticas te dan una estructura inicial, pero cuando surgen problemas de despliegue (como configurar las rutas correctas para servidores como GitHub Pages), errores de sintaxis o cuando simplemente se quiere personalizar el diseño para que quede exactamente como uno quiere, el criterio humano es indispensable.
Si no entendés el fondo de tu propio código, no hay forma de corregirlo ni de escalarlo.
